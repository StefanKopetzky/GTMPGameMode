
1. Copy ..\GameMode\GT-MP\GTMPVoiceServer.dll to your GT-MP Server directory
2. Copy all files to resources\PureVoice of your GT-MP Server
3. edit your settings.xml , add 
  <resource src="PureVoice" />
4. edit resources\PureVoice\meta.xml
5. (players/Users) need to install the ts3-Plugin ( ..\GameMode\GT-MP\GTMPVoice_*.ts3_plugin )
